Everything you need for AI-augmented knowledge work

Built on Kilo Code's proven agentic architecture — adapted for the Obsidian knowledge worker.

🔧

30+ Vault Tools

Read, write, search, edit frontmatter, navigate links, create Canvas files, manage Bases databases, and more — all with model approval.

🔍

Hybrid Semantic Search

Local vector index (Vectra + Xenova) combined with full-text search via Reciprocal Rank Fusion. No cloud required — your notes stay private.

⚡

Agent Modes

Built-in Ask (read-only) and Agent (full) modes. Create custom modes with their own roles, tool sets, model overrides, and instructions.

🤝

Multi-Agent Workflows

Spawn sub-agents with new_task for orchestrator-worker, prompt chaining, evaluator-optimizer, and routing patterns.

📋

Context Injection

Rules, Skills, and Workflows let you customize agent behavior. Define permanent instructions, keyword-matched skills, and slash-command templates.

🛡️

Safety by Default

Every write operation requires approval. Automatic isomorphic-git checkpoints before each task. Audit log. Ignore and protected file patterns.

🌐

Web Research

Fetch any URL as Markdown. Search the web via Brave or Tavily. Combine external information with your vault knowledge seamlessly.

🔌

MCP Integration

Connect MCP servers via stdio, SSE, or HTTP. Tools are auto-discovered and exposed to the agent. Per-mode server whitelisting.

🤖

Multi-Provider

Anthropic, OpenAI, Ollama, LM Studio, OpenRouter, Azure — or any OpenAI-compatible API. Per-mode model overrides.
